## How to Use 🛠️

1. **Instagram Download**:
   - Paste the Instagram video link in the input field.
   - Click "Download" to fetch the video.
   - Click "Download Now" to save the video.

2. **YouTube Download**:
   - Paste the YouTube video link in the input field.
   - Click "Download" to fetch the video.
   - Click "Download Now" to save the video.
